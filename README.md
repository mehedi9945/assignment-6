# DigiTools

A modern React website for a digital tools marketplace that includes product browsing, cart management, pricing plans, and responsive landing page design.

## Description

DigiTools is a buying website for premium digital products. It features a polished landing page, a product catalog with add-to-cart functionality, a responsive checkout section, and toast notifications for user actions.

## Technologies Used

- React.js
- Tailwind CSS
- JavaScript (ES6+)
- React Toastify
- JSON product data

## Features

- Interactive navbar with cart item count
- Product and cart toggle view with responsive design
- Add to cart, remove items, and proceed to checkout behavior
- Toast notifications for cart actions
- Pricing cards, steps section, and footer matching the assignment layout

## Run Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open the app in the browser at the Vite URL.
